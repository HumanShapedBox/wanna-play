***Персонажи:
Арбалетчик, Маг, Крестьянин, Вор
Снайпер, Монах, Крестьянин, Копьеносец
NB! в скобках указаны базовые значения или ключевые особенности

*Команда 1

Арбалетчик: 
Свойства:
Урон(15), здоровье(100), имя, меткость(рандом: 0 - мимо, 1 - попал*),
местоположение, кивер с болтами (10)
* меткость и степень урона можно увеличить с помощью магии
Действия:
Стрельба, защита (рандом: 0 - пропустил удар, 1 - защитился от удара),
отравленный болт (бьёт на поражение, появленияется при соблюдении ряда условий: 
                  маг в команде имеет более 50хр, а сам герой не промахивался 5 раз подряд)
* защиту можно улучшить с помощью магии

Вор: 
Свойства:
Урон(10), дальность(1-2), здоровье(100), имя, меткость(рандом: 0 - мимо, 1 - попал*),
местоположение
* меткость и степень урона можно увеличить с помощью магии
Действия:
Атака, двойной урар (урон увеличивается вдвое, 
если в течение 3 ходов вор не получал повреждений и не промахивался),
перемещение (аналогично шахматному королю), 
защита (рандом: 0 - пропустил удар, 1 - защитился от удара*),
* защиту можно улучшить с помощью магии

Маг: 
Свойства:
Здоровье(100), имя, урон(20*), дальность(1-8), меткость(всегда 1), местоположение, 
магия(100)
*маг способен самостоятельно атаковать противника только в случае, 
если остальные персонажи его команды не способны этого сделать
Действия:
атака, лечение(+10хр всем игрокам за ход), пополнение магии,
увеличение силы и меткости(при использовании член команды, совершающий следующий ход,
не промахивается и получает +5 к урону)

Крестьянин: 
Свойства:
Урон(0*), дальность(1-2), здоровье(100), имя, меткость(рандом: 0 - мимо, 1 - попал),
местоположение, обоз с боеприпасами
* в случае, если в команде не осталось боеспособных персонажей, крестьянин 
способен вступить в бой и получает урон - 10
* меткость и степень урона можно увеличить с помощью магии
Действия:
атака(как у шахматного короля)
